# WorldMask - Game Jam 2026 (Theme: MASK)

## About
A game made for **Game Jam 2026** with the theme **MASK**.

You play through multiple levels using **layers** to take advantage of obstacles. Switching layer changes what is visible and creates new path.

## Key feature
- Layer-based
- Mask mechanic: layer switching
- Each level introduces new tricks and challenges

## Controls
- WASD: Move
- Space: Jump
- Mouse click: Toggle layers

## Tech 
- Engine: Unity
- Language: C#
